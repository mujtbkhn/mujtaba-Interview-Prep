Flex is a technique introduced in modern css to make the model follow a responsive layout structure without using the float or positioning, basically flex is used to make "Flexable ie responsive things"
    Flex has two directions:
  • Row : which aligns items in rows from left to right
  • Column: which aligns items in columns from top to bottom

By default Flex is in row direction

Flex has two axis's
  • Main Axis
  • Cross Axis

    • As flex is in row by default so the main axis is row and the cross axis is column , there are two things
  • Justify content : which justifies the contents of flex in the main axis
  • Align Items : which aligns the items of flex in the cross axis
